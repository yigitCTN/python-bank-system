# python-bank-system

Bu proje, Python öğrenme sürecimde geliştirdiğim ilk mini banka sistemi projesidir.

Projede kullanıcıların terminal üzerinden çeşitli banka işlemlerini yapabilmesi amaçlanmıştır.

## Özellikler

* Bakiye görüntüleme
* Günlük limit ayarlama
* Para yatırma
* Para çekme
* Para transferi
* İşlem geçmişi görüntüleme
* Güvenli çıkış sistemi

## Kullandığım Konular

* if / elif / else
* while döngüsü
* for döngüsü
* listeler
* kullanıcıdan veri alma
* koşullu ifadeler
* break kullanımı

## Not

Bu proje tamamen öğrenme amaçlı yapılmıştır. İleride yeni bilgiler öğrendikçe projeyi geliştirmeyi düşünüyorum.
